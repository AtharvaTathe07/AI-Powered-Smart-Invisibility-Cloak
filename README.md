# AI-Powered Smart Invisibility Cloak

## Overview

AI-Powered Smart Invisibility Cloak is an advanced Computer Vision project that creates a real-time invisibility effect using intelligent object detection, tracking, and dynamic background replacement techniques.

The system detects a colored cloak, tracks its movement across frames, and seamlessly replaces the detected region with background information to create the illusion of invisibility.

---

## Key Features

### Computer Vision

* Real-time video processing
* HSV color-based object detection
* Dynamic image masking
* Background subtraction

### AI Components

* YOLO-based object detection architecture
* Object tracking framework
* Dynamic background adaptation
* Noise reduction and mask optimization

### Performance Enhancements

* Adaptive background modeling
* Morphological image processing
* Frame stabilization
* Real-time rendering pipeline
