# AI-Powered Smart Invisibility Cloak

## Overview

AI-Powered Smart Invisibility Cloak is a Computer Vision project that combines object detection, tracking, image masking, and dynamic background replacement techniques to create a real-time invisibility effect.

This project demonstrates core concepts of Artificial Intelligence, Image Processing, and Computer Vision using Python and OpenCV.

---

## Features

- Real-time invisibility cloak simulation
- HSV color-based cloak detection
- Object tracking architecture
- YOLO detection module structure
- Dynamic background replacement
- Morphological image processing
- Modular project design

## Workflow

1. Capture Input Frame
2. Detect Cloak Region
3. Generate Binary Mask
4. Track Cloak Movement
5. Replace Region With Background
6. Generate Final Output
